¿Que es html?



CSS son las siglas en inglés de Cascading Style Sheets, que en español se traduce como "hojas de estilo en cascada". Es un lenguaje de diseño web que se utiliza para dar estilo a páginas web. Características Permite controlar el color, la fuente, el tamaño, la posición, los márgenes, las animaciones y los efectos visuales de los elementos de una página web Separa el contenido del diseño, lo que facilita la gestión visual de las páginas web Permite que varias páginas compartan la misma hoja de estilo Facilita el mantenimiento de las páginas web Aumenta la velocidad de carga de las páginas
